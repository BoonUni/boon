**บุญ** คือ ชุดตัวอักษรตระกูลหนึ่ง (font family) ที่มุ่งแสดงผลการจัดเรียงตัวอักษรไทยให้ถูกต้องในเว็บบราวเซอร์

อ่านรายละเอียดเพิ่มเติมเกี่ยวกับการใช้งานบุญฟอนต์ได้ที่ [Boon Project Page](http://sungsit.com/boon/)
